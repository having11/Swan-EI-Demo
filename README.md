## Hardware

This project requires just three components:

- A Swan Development Board
- An LSM9DS1 IMU
- An SSD1306 128x64 OLED screen

Simply wire them as follows:

| Swan  | LSM9DS1 | OLED  |
| :---: | :-----: | :---: |
|  SCL  |   SCL   |  SCL  |
|  SDA  |   SDA   |  SDA  |
|  3V3  |   VDD   |  VCC  |
|  GND  |   GND   |  GND  |
